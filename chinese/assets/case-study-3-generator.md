# 场景三：简单生成任务

## 触发词
「生成周报」「生成报告模板」「格式固定」

## 特点
- 不是 Pipeline！是 Generator！
- 任务简单，不需要多步骤
- 直接填充模板

## 决策树判断

```
任务 = 多步骤？
    │
    ├── 否 → Generator（直接生成）
    │
    └── 是 → Pipeline + WORKFLOW
```
